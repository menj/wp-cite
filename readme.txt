Help readers cite your content properly with dynamic references

== Description ==

**Originally developed by EnigmaWeb**, now maintained and improved by menj. Display professional citations using the shortcode [cite] with dynamic template tags.

= Credits =
* Original codebase: EnigmaWeb (2014-2022)
* Current maintenance & modernization: menj

= Key Features =
* Modern & secure codebase
* 6 dynamic tags: {author}, {title}, {sitename}, {date}, {publication_date}, {permalink}
